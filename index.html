Automated Curry Maker

Fusion 360 Mechanism Design Automation

The Problem

Cooking consistently requires precise timing and stirring. Manual methods are labor-intensive.

The Solution

I designed a smart cooking machine that automates the entire process using low-cost mechanical components.

Key Mechanisms

  • Rotating Spoon: A custom gear-driven assembly to ensure even mixing without burning.
  • Slotted Lid Timing: An automated ingredient-release system controlled by electronic timing.
  • Cost Optimization: Built using basic electronic controls rather than expensive PLCs.
